<blockquote> y <cite>

EL elemento <blockquote> indica que el texto encerrado es una cita extensa, normalmente se coloca un URL con la fuente de la cita usando el atributo cite, mientras que la representacion tectual de la fuente puede darse usando la etiqueta <cite>.

<q>

El elemento <q> indica que el texto encerrado es una cita en linea corta, la mayoria de los navegadores actuales encierra entre comillas el contenido. Al igual que en <blockquote>, se usa el atributo <cite> para indicar el URL de la cita.

<abbr>

<sub> y <sup>

Elemento <sub> y <sup> son usados para representar subindicies y superindices respectivamente, estos solo son usados por razonas tipograficas (de como se ve el texto)

Un ejemplo seria:

a2 + b2 = c2
Almost every developer's favorite molecule is C8H10N4O2, also known as "caffeine."

La etiqueta <code> es usada para mostrar sus contenidos con un estilo que muestre que el texto es un fragmento pequeño de codigo para computadora.

The push() method adds one or more elements to the end of an array and returns the new length of the array.

<var>

El elemento <var> representa el nombre de una variable en una expresion matemática o un contexto de programacion.

Un ejemplo seria:

The volume of a box is l × w × h, where l represents the length, w the width and h the height of the box.

<kbd>

El elemento <kbd> representa una parte de texto en linea que es una entrada de teclado, una entrada de voz o cualquier otro dispoditivo de entrada de texto.

Un ejemplo seria:

Please press Ctrl + Shift + R to re-render an MDN page.

<samp>

El elemento <samp> es usado para encerrar texto en linea que representa una muestra de la salida de un de un programa de computadora.

Un ejemplo seria:

I was trying to boot my computer, but I got this hilarious message:
Keyboard not found
Press F1 to continue

<time>

La etiqueta <time> representa un periodo especifico de tiempo, este puede incluir el atributo datetime ek cyak traduce las fechas a un formtato legible para programas de computadora, permitiendo mejores resultados en motores de busqueda o caracteristicas especiales como alarmas.

Un ejemplo seria:

The concert starts at 20:00 and you'll be able to enjoy the band for at least 2h 30m.
